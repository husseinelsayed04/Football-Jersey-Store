# ⚽ Football Jersey Store

A modern Android e-commerce application for football jersey shopping, built with **Java** and **XML** using **Android Studio**.

The application provides a clean and intuitive shopping experience, allowing users to browse football jerseys, customize products, manage their shopping cart, and complete purchases seamlessly.

---

## ✨ Features

- 🔐 User Authentication
- 🏠 Home Screen
- 📂 Browse Categories
- 👕 Football Jersey Catalog
- 🔍 Product Details
- 🎨 Jersey Customization
- 🛒 Shopping Cart
- 💳 Checkout
- ✅ Order Confirmation

---

## 🛠️ Tech Stack

- Java
- XML
- Android Studio
- - ConstraintLayout
- CardView
- Material Components
- Intent Navigation
