# ⚽ Football Jersey Store

A modern Android e-commerce application for football jersey shopping, built with **Java** and **XML** using **Android Studio**. The application provides a clean and intuitive shopping experience, allowing users to browse jerseys, customize products, manage their cart, and complete purchases.

---
## ✨ Features

- User Login
- Browse Football Jerseys
- Categories
- Product Details
- Favorite Jerseys
- Shopping Cart
- Jersey Customization
- Checkout
- Order Confirmation
- Clean UI Design

---

## 🛠️ Built With

- Java
- XML
- Android Studio
- CardView
- ConstraintLayout
- Material Components
- Intent Navigation
