# ⚽ Football Jersey Store

A modern Android e-commerce application for football jersey shopping, built with **Java** and **XML** using **Android Studio**. The application provides a clean and intuitive shopping experience, allowing users to browse football jerseys, customize products, manage their cart, and complete purchases.

---

## ✨ Features

- 🔐 User Login
- 🏠 Home Screen
- 📂 Browse Categories
- 👕 Football Jersey Catalog
- 🔍 Product Details
- ❤️ Favorite Jerseys
- 🎨 Jersey Customization
- 🛒 Shopping Cart
- 💳 Checkout
- ✅ Order Confirmation
- 🎯 Modern & User-Friendly UI

---

## 🛠️ Tech Stack

- Java
- XML
- Android Studio
- RecyclerView
- ConstraintLayout
- CardView
- Material Components
- Intent Navigation
